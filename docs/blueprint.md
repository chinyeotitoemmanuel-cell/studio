# **App Name**: AfriStory Hub

## Core Features:

- User Authentication: Secure user registration and login with Google, Facebook, email, OAuth 2.0 and JWT.
- Pro and Draft Content: Differentiated content areas for professional and amateur artists, allowing users to explore diverse African storytelling.
- Interactive Reader: Webtoon and BD reader for an immersive experience (scroll & page)
- AI Moderation: Use AI as a tool to automatically detect hate speech, sexually explicit content and violence in community uploads.
- Internal Currency: AfriCoins will provide virtual currency that allows readers to reward creators by purchasing content and tipping artists.
- Draft Tier: Community feature where new creators can get feedback and recognition. Pro creators can be mentors and identify promising talent. Use AI to suggest constructive, non-negative feedback to new creators.
- Content Posting: Creators are able to upload works, with the system supporting single image uploads for smaller posts and multi-part uploads (eg: multi-page PDF documents).

## Style Guidelines:

- Primary color: Gold (#f2a60d) to represent African heritage and artistic value.
- Background color: Dark Brown/Charcoal (#221c10) for a sophisticated dark mode experience.
- Accent color: Red-Orange (#c2410c) to highlight important interactive elements.
- Body and headline font: 'Inter', sans-serif, for a modern, readable interface.
- Use Material Symbols Outlined for consistent and clear visual communication.
- Mobile-first design with a focus on fluid, intuitive navigation.
- Subtle transitions and animations to enhance user engagement without being distracting.
